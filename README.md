# 🥦 FreshGrocer - Premium Organic Grocery Delivery

FreshGrocer is a modern, high-performance landing page for a premium grocery delivery service. Rebuilt from the ground up using **React**, **Vite**, and **Tailwind CSS**, it features a stunning glassmorphic design, fluid animations, and a focus on user experience.

## ✨ Features

- **💎 Premium Glassmorphic UI**: Transparent, blurred interfaces for a modern, high-end feel.
- **🚀 Dynamic Animations**: Smooth reveal-on-scroll and micro-interactions powered by **Framer Motion**.
- **📱 Fully Responsive**: Optimized for every device, from mobile phones to high-resolution desktops.
- **🎨 Custom Design System**: A nature-inspired color palette and typography (Outfit & Inter).
- **🛒 Interactive Components**: 
  - Glassmorphic fixed navigation.
  - Animated hero section with floating status elements.
  - Hover-aware feature cards.
  - High-impact CTA and comprehensive footer.

## 🛠️ Tech Stack

- **Framework**: [React 19](https://react.dev/)
- **Build Tool**: [Vite 5](https://vitejs.dev/)
- **Styling**: [Tailwind CSS v3](https://tailwindcss.com/)
- **Animations**: [Framer Motion](https://www.framer.com/motion/)
- **Icons**: [Lucide React](https://lucide.dev/)
- **Fonts**: [Google Fonts (Outfit & Inter)](https://fonts.google.com/)

## 🚀 Getting Started

### Prerequisites

- [Node.js](https://nodejs.org/) (v18.18 or higher recommended)
- [npm](https://www.npmjs.com/)

### Installation

1. **Clone the repository**:
   ```bash
   git clone https://github.com/obarakoo/FreshGrocer.git
   cd FreshGrocer
   ```

2. **Install dependencies**:
   ```bash
   npm install
   ```

3. **Start the development server**:
   ```bash
   npm run dev
   ```

4. **Build for production**:
   ```bash
   npm run build
   ```
